# Followup Coach

A minimal follow-up webapp to improve sales execution and customer intimacy.

## What it does

- Accounts with A/B/C tiers and value ranking
- Today view with follow-up cadence (A=7d, B=14d, C=30d)
- Quick Log: every interaction must include next step + date
- Weekly Pack: text summary ready to copy/paste

## Tech stack

- Next.js (App Router)
- Supabase (Auth + Postgres + RLS)
- Vercel deploy

## Local setup

1. Install deps

```bash
npm install
```
